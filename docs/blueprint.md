# **App Name**: LinkedUp

## Core Features:

- LinkedIn Post Generation: Generate engaging LinkedIn posts based on topic and tone, enhanced by generative AI, acting as a writing tool that strategically decides if and how to include various informational inputs.
- Dashboard Overview: Display a dashboard summarizing total posts, and provide navigation to the generator, calendar, and library pages.
- Content Calendar: Enable users to schedule and plan their LinkedIn posts in a calendar view.
- Post Library: Allow users to view, regenerate, edit, and delete previously generated LinkedIn posts.
- Firebase Authentication: Implement user authentication using Firebase, managing user accounts and access.
- Free Tier & Premium Upgrade: Offer 5 free posts for new users with an option to upgrade to a premium plan for enhanced post generation models.
- Razorpay Payment Integration: Integrate Razorpay for handling premium subscription payments.

## Style Guidelines:

- Primary color: Deep sky blue (#38b6ff) for a professional and trustworthy feel.
- Background color: Light grayish-blue (#f0f8ff) to ensure a clean and modern look.
- Accent color: Electric purple (#bf40bf) for call-to-action buttons.
- Body and headline font: 'Inter' sans-serif for a clean and readable user experience.
- Use simple, professional icons from a library like FontAwesome for navigation and actions.
- Maintain a clean, organized layout with a focus on user-friendly navigation and content presentation.
- Use subtle transitions and animations for a polished and engaging user experience.
